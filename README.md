# <h1 align= "center">Sistema de Registro de Estacionamiento</h1>

<h3> 🛠 Este sistema está desarrollado con las tecnologías siguientes: Node.js, Express.js, MySQL, EJS, HTML5, CSS3 y Bootstrap</h3>
<hr>

[![estacionamiento.png](https://i.postimg.cc/x1Q6sq4D/estacionamiento.png)](https://postimg.cc/0btGj5vc)

<hr>

### <h3> :computer: Este Proyecto cuenta con las siguientes funciones: </h3>

<p aligth="justify"> ⭐️ Función de Registro de Usuarios: Los usuarios para registrarse deben ingresar los siguientes datos: nombre completo, un nombre de usuario y una contraseña.</p>

<p aligth="justify"> ⭐️ Función de Inicio de Sesión: El usuario debe iniciar sesión con su nombre de usuario y su contraseña de registro.</p>

<p aligth="justify"> ⭐️ Función de Registro de Autos: Podrá registrar los datos del usuario que registra su auto, como también su cédula, marca de auto, placa del auto y la hora de ingreso y salida del estacionamiento.</p>

<p aligth="justify"> ⭐️ Función listar autos registrados: En una tabla se podrá visualizar la información de los autos registrados con su respectivo valor a cancelar por el estacionamiento.</p>

<p aligth="justify"> ⭐️ Función de Editar los registros de los autos: Se podrá editar la información de los registros de los autos ingresados previamente. </p>

<p aligth="justify"> ⭐️ Función de Eliminar los registros de los autos: Se podrá eliminar cualquier registro realizado. </p>

<p aligth="justify"> ⭐️ Tarjetas de Registro de Autos: Indica cierta información de los registros de los autos como el nombre del propietario, el código de registro, la placa y la marca del auto y el precio a pagar por el uso del estacionamiento. </p>


</br>

### <h3> ⚡ Instalación del proyecto en su ordenador: </h3>
- Clonar el proyecto en cualquier directorio de su preferencia.
- Abrir el proyecto en un editor de código de su preferencia. 
- Abrir un terminal desde el directorio donde se encuentra su proyecto y procede a instalar las dependencias del proyecto con el comando npm install
- Dentro del proyecto existe el archivo llamado script_bdd.txt, ahí podrá encontrar la base de datos la cual deberá ser creada en MySQL. 
- Una vez instaladas las dependencias y creada la base de datos en MySQL podrá ejecutar el proyecto en una terminal con el comando npm run dev.
